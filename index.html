Política de Privacidade – ReminderTimer

Última atualização: 02/03/2026

O aplicativo ReminderTimer respeita sua privacidade. Esta Política de Privacidade explica quais dados podem ser coletados, como são utilizados e quais são seus direitos. Ao utilizar o aplicativo, você concorda com os termos descritos abaixo. 1. Sobre o Aplicativo O ReminderTimer é um aplicativo de temporizador ativo com lembretes intermediários e alarme em tela cheia persistente. Ele funciona principalmente offline e não exige criação de conta. 2. Dados Coletados O ReminderTimer não coleta dados pessoais diretamente, como: Nome CPF Endereço Contatos Localização Arquivos pessoais Entretanto, serviços de terceiros integrados ao aplicativo podem coletar dados automaticamente. 3. Publicidade (Google AdMob) O aplicativo utiliza anúncios fornecidos pelo Google AdMob. O AdMob pode coletar automaticamente: Identificadores de dispositivo Dados de uso Endereço IP Informações de interação com anúncios Esses dados são utilizados para: Exibir anúncios relevantes Medir desempenho de anúncios Prevenir fraudes O ReminderTimer não controla diretamente a coleta feita pelo AdMob. Você pode consultar a política do Google em: https://policies.google.com/privacy 4. Compras dentro do Aplicativo O aplicativo oferece uma compra única para remover anúncios, processada através do Google Play Billing. As transações são processadas exclusivamente pelo Google. O ReminderTimer não tem acesso a: Dados do cartão Informações bancárias Dados financeiros 5. Consentimento (LGPD / GDPR) Quando exigido por lei, o aplicativo exibirá uma mensagem de consentimento para coleta de dados relacionados à publicidade. O usuário pode: Aceitar Recusar Gerenciar preferências A qualquer momento, é possível redefinir o consentimento nas configurações do dispositivo. 6. Dados Armazenados Localmente O ReminderTimer armazena apenas dados no próprio dispositivo do usuário, como: Configurações do temporizador Preferências de som e vibração Histórico de timers concluídos Status de remoção de anúncios Esses dados não são enviados para servidores externos. 7. Segurança O aplicativo utiliza as práticas recomendadas pela plataforma Android para proteger dados locais. Nenhum dado sensível é transmitido para servidores próprios. 8. Direitos do Usuário (Brasil – LGPD) De acordo com a Lei Geral de Proteção de Dados (LGPD), o usuário tem direito a: Saber quais dados são coletados Solicitar exclusão de dados armazenados Revogar consentimento Como o ReminderTimer não armazena dados pessoais em servidores próprios, a maioria das solicitações pode ser resolvida apagando os dados do aplicativo nas configurações do Android. 9. Alterações nesta Política Esta Política pode ser atualizada periodicamente. Recomenda-se revisar esta página ocasionalmente. 10.
