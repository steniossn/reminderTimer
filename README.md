
# ReminderTimer – Política de Privacidade

Este repositório hospeda a Política de Privacidade oficial do aplicativo ReminderTimer, publicada publicamente para atender às exigências da:

Google Play Store

Google AdMob

Google Play Billing

Sobre o aplicativo

ReminderTimer é um temporizador ativo com:

Contagem em segundo plano (Foreground Service)

Lembretes intermediários

Alarme persistente em tela cheia

Histórico local via banco de dados interno

O aplicativo funciona principalmente offline e não exige criação de conta.

Publicidade

O app utiliza anúncios fornecidos pelo Google AdMob.

Usuários podem optar por remover os anúncios através de compra única dentro do aplicativo.

Coleta de Dados

O ReminderTimer não coleta dados pessoais diretamente.
Serviços de terceiros (como AdMob) podem coletar dados técnicos automaticamente para fins de publicidade.
